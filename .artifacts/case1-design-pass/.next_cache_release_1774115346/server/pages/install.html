Install OpenCrab

在 macOS 上启动 OpenCrab 的最短路径。

先确认环境，再走最短链路。别把安装过程搞成研究项目，先跑起来才有资格谈后面的工作流。

当前推荐前提

  • 推荐环境：macOS
  • Node.js >= 20.9
  • 本机 `codex login status` 可用

Quickstart

先照这个顺序跑通，再扩展渠道、任务和技能。

01

确认 Codex 登录状态

OpenCrab 当前核心对话能力依赖本机 Codex 登录状态，这一步别装糊涂跳过。

codex login
codex login status
02

准备运行环境

在 macOS 上安装 Node.js，再装 Codex CLI，后面就别用奇怪环境折腾自己。

brew install node
npm i -g @openai/codex
03

拉取代码并安装依赖

拿到仓库后先装依赖，别一上来就改配置。

git clone https://github.com/KetteyMan/opencrab.git
cd opencrab
npm install
04

初始化本地配置

复制默认环境文件。大多数首次安装场景，不需要先乱改环境变量。

cp .env.example .env.local
05

启动开发环境

先把网页跑起来，再继续接渠道、任务或团队能力。

npm run dev
06

做最短验证

打开设置页确认账户连接状态正常，再开始新建对话。

open http://127.0.0.1:3000/settings

Before you proceed

OpenCrab 更适合想把 AI 接回本地工作流的人。

如果你期待的是纯在线开箱即用 SaaS，这页已经替你筛掉预期偏差了。反过来，如果你就是想要一个本地优先的工作台，这个方向就对。
