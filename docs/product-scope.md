## Positioning

`OpenCrab` 是一个以中文为主、面向普通用户的本地优先 AI 工作台。

产品主入口是聊天，对话体验参考 ChatGPT 风格工作台；底层执行能力复用 agent 能力，但产品形态不做成纯 CLI。

## Current V1 Focus

- 聊天主入口
- 历史对话与文件夹管理
- 智能体列表、详情与自定义创建
- Team Mode 团队创建与 Team Room 推进
- 定时任务创建、暂停 / 恢复、立即执行与结果对话
- 图片与文件上传
- Codex 模型与推理强度选择
- 默认语言选择
- 浏览器工具接入
- Skills 目录浏览、详情查看与本地状态管理
- 设置页中的默认模型、推理强度、语言、权限模式与浏览器模式
- 关于我们页面中的产品介绍、原则、路线图与联系入口

## Channels V1

`Channels` 已进入第一版实现，当前范围是：

- Telegram bot webhook 入站
- 飞书长连接入站
- 文本消息去重
- Telegram 图片 / 文件下载到 OpenCrab 附件存储
- 远程 chat 与 OpenCrab conversation 自动绑定
- 渠道内文本回复回推

暂不包含：

- 飞书图片 / 文件消息闭环
- 主动群发
- 多租户与权限体系
- 复杂运营面板

## Tasks V1

`定时任务` 当前已经可用，范围是：

- 任务页创建 / 查看 / 暂停 / 恢复 / 立即执行
- 任务结果自动回流到专属对话
- 从当前对话直接创建定时任务

暂不包含：

- 系统级后台常驻调度
- 复杂 cron 表达式
- 渠道内主动推送任务结果

## Product Principles

- 默认简单，不让普通用户先学术语
- 已有 Codex 能力优先直接使用
- 少而稳，先把聊天主链路做扎实
- 中文优先
- 设置低频化
- 本地优先
- 从“产品工作台”角度统一对话、执行、任务、渠道和协作
