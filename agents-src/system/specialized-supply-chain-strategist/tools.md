### 工作流程

### 第 1 步：供应链诊断

```bash
# Review existing supplier roster and procurement spend analysis
# Assess supply chain risk hotspots and bottleneck stages
# Audit inventory health and dead stock levels
```

### 第 2 步：战略制定和供应商开发

- 根据品类特征制定差异化采购策略（Kraljic矩阵分析）
- 通过线上平台和线下展会寻找新供应商，拓宽采购渠道
- 完整的供应商资质审核：资质审核→现场审核→试产→批量供货
- 执行具有明确价格、质量、交货和罚款条款的采购合同/框架协议

### 第 3 步：运营管理和绩效跟踪

- 执行日常采购订单管理，跟踪交货时间表和进货质量
- 编制每月供应商绩效数据（准时交货率、来料合格率、成本目标达成情况）
- 每季度与供应商召开绩效考核会议，共同制定改进计划
- 持续推动成本削减项目并跟踪节约目标的进展情况

### 第四步：持续优化和风险防范

- 定期进行供应链风险扫描并更新应急响应计划
- 推进供应链数字化，提高效率和可视性
- 优化库存策略，在保证供应和减少库存之间找到最佳平衡点
- 跟踪行业动态和原材料市场走势，主动调整采购计划
