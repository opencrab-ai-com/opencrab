### 工作流程

### 第一阶段：发现和技术基础
1. **技术审核**：抓取网站（Screaming Frog / Sitebulb 等效分析），识别可抓取性、索引和性能问题
2. **搜索控制台分析**：查看索引覆盖范围、手动操作、核心网络生命周期和搜索性能数据
3. **竞争格局**：确定排名前 5 的自然竞争对手、他们的内容策略和链接配置文件
4. **基线指标**：记录当前的有机流量、关键字位置、域名权限和转化率

### 第二阶段：关键词策略和内容规划
1. **关键字研究**：构建按主题集群和搜索意图分组的全面关键字宇宙
2. **内容审核**：将现有内容映射到目标关键字，识别差距和蚕食
3. **主题集群架构**：通过内部链接策略设计支柱页面和支持内容
4. **内容日历**：根据影响潜力（数量×可实现性）确定内容创建/优化的优先顺序

### 第三阶段：页面和技术执行
1. **技术修复**：解决关键抓取问题、实施结构化数据、优化核心 Web Vitals
2. **内容优化**：更新现有页面，改进定位、结构和深度
3. **新内容创建**：针对已发现的差距和机会制作高质量的内容
4. **内部链接**：构建将集群连接到支柱的上下文内部链接架构

### 第四阶段：权威建设和页外
1. **链接配置文件分析**：评估当前反向链接的健康状况并识别增长机会
2. **数字公关活动**：创建可链接资产并执行记者/博主外展活动
3. **品牌提及监控**：转换未链接的提及并管理在线声誉
4. **竞争对手链接差距**：识别并寻找竞争对手拥有但我们没有的链接源

### 第五阶段：测量和迭代
1. **排名跟踪**：每周监控关键词位置，分析移动模式
2. **流量分析**：按着陆页、意图类型和转化路径细分有机流量
3. **投资回报率报告**：计算自然搜索收入归因和每次获取成本
4. **策略细化**：根据算法更新、性能数据和竞争变化调整优先级
