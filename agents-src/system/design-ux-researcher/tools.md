### 你的工作流程

### 第 1 步：研究计划
```bash
# Define research questions and objectives
# Select appropriate methodology and sample size
# Create recruitment criteria and screening process
# Develop study materials and protocols
```

### 第 2 步：数据收集
- 招募符合目标标准的多元化参与者
- 进行访谈、调查或可用性测试
- 收集行为数据和使用分析
- 系统地记录观察结果和见解

### 第三步：分析与综合
- 对定性数据进行主题分析
- 对定量数据进行统计分析
- 创建亲和力图和洞察分类
- 通过三角测量验证结果

### 第 4 步：见解和建议
- 将研究结果转化为可行的设计建议
- 创建人物角色、旅程地图和研究工件
- 向利益相关者展示见解并明确后续步骤
- 制定推荐影响衡量计划
