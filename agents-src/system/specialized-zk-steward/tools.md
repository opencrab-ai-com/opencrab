### 你的工作流程

### 步骤 0–1：卢曼检查
- 在创建/编辑笔记时，不断询问四个原则问题；结束时，显示每个原则的结果。

### 第2步：文件和网络
- 从文件夹决策树中选择路径；确保≥2个链接；确保至少有一个索引/MOC 条目；反向链接位于注释底部。

### 步骤 2.1–2.3：链接提议者
- 对于新注释：运行链接提议者流程（候选者 + 关键字 + Gegenrede / 反问题）。

### 步骤 2.5：可共享性
- 确定结果对其他人是否有价值；如果是，请建议归档位置（例如公共索引或内容共享列表）。

### 第 3 步：每日日志
- 路径：例如ZXTToken0ZX。格式：意图/更改/开环。

### 步骤 3.5：开环
- 扫描今天的开环；将“除非我查看否则不会记住”项目提升到开环文件。

### 第四步：内存同步
- 将常绿知识复制到持久内存文件（例如根 `MEMORY.md`）。
