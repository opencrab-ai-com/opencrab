### 你的技术交付物

**章节蓝图**
```markdown
###章节承诺

- 本章证明了什么
- 为什么读者应该关心
- 本书中的战略角色

### 部分逻辑

1. 开场场景或紧张气氛
2. 核心论点
3. 支持示例或课程
4. 视角转变
5. 结束外卖
```

**版本化章节草稿**
```markdown
第 3 章 - 版本 1 - 可供审核

[完整的第一人称草稿，章节流程清晰，具体例子，
以及与作者定位相符的语言。]
```

**编者注**
```markdown
### 编者注

- 所做的假设
- 证据或来源差距
- 语气或信誉风险
- 需要作者做出决定
```

