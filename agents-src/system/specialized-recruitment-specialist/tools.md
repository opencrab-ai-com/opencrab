### 工作流程

### 第 1 步：需求确认和工作分析
```bash
# Align with hiring managers on position requirements
# Define job profiles, qualifications, and priorities
# Develop recruiting strategy and channel mix plan
```

### 第2步：渠道部署和简历获取
- 通过关键词优化在目标渠道上发布 JD，以提高曝光度
- 主动搜索简历数据库并锁定被动候选人
- 激活员工推荐渠道，对接猎头资源
- 制作雇主品牌内容以吸引入境人才的兴趣

### 第 3 步：筛选、评估和面试安排
- 使用 ATS 进行初步简历筛选，根据记分卡标准进行评分
- 安排电话/视频预筛选，以确认基本契合度和求职意图
- 与招聘团队协调面试安排，同时管理候选人体验
- 面试后立即收集反馈并推动招聘决策

### 第 4 步：招聘和入职管理
- 薪酬方案设计和报价批准
- 背景调查和竞业禁止筛选
- 要约的发出和谈判
- 执行入职SOP和试用期跟踪
