### 工作流程

### 第一阶段：品牌生活方式定位
1. **受众深度剖析**：人口统计、兴趣、生活方式愿望、痛点
2. **生活方式叙事发展**：品牌故事、价值观、审美个性、独特定位
3. **审美框架创建**：摄影风格（极简/极简）、滤镜偏好、色彩心理学
4. **竞争格局**：分析品类中的顶级生活方式品牌，识别差异化机会

### 第二阶段：内容策略和日历
1. **热门话题研究**：每周趋势分析、即将到来的季节性机会、病毒式内容模式
2. **内容组合规划**：70%生活方式，20%潮流参与，10%产品/品牌推广平衡
3. **内容支柱**：定义符合品牌和受众兴趣的 4-5 个核心内容类别
4. **内容日历**：30 天滚动日历，包含计时、趋势整合、主题标签策略

### 第三阶段：内容创建和优化
1. **微内容制作**：高效的内容创建系统可实现一致的输出（每周 10 个以上帖子的容量）
2. **视觉一致性**：在所有内容中一致地应用美学框架
3. **文案优化**：情感挂钩、趋势相关语言、战略性 CTA 放置
4. **技术优化**：图像格式（9:16优先）、视频长度（15-60秒最佳）、主题标签放置

### 第四阶段：社区建设和发展
1. **积极参与**：评论热门帖子，2小时内回复社区
2. **影响者合作**：与微型影响者（10k-100k 关注者）合作以实现真实的放大
3. **UGC 活动**：品牌主题标签挑战、客户特色计划、社区共同创建
4. **数据驱动迭代**：每周绩效分析、趋势适应、受众反馈纳入

### 第 5 阶段：性能分析和扩展
1. **每周绩效回顾**：表现最佳的内容分析、热门话题的有效性
2. **算法优化**：发布时间细化、主题标签性能跟踪、参与模式分析
3. **转化跟踪**：链接点击跟踪、电子商务集成、下游指标测量
4. **扩展策略**：识别病毒式内容模式、扩展成功的内容系列、平台扩展
