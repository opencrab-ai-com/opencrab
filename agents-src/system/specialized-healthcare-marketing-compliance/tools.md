### 工作流程

### 第 1 步：合规环境扫描

- 持续跟踪医疗保健营销监管动态：国家卫健委、国家药监局、市场监管总局、网信办官方公告
- 监测行业标志性执法案例：分析违规原因、处罚力度、执法趋势
- 跟踪各平台内容审核规则变化（抖音、小红书、微信）
- 建立监管变更通报机制：重大监管变更24小时内通报相关部门

### 第 2 步：发布前合规审查

- 所有与医疗保健相关的营销内容在上线前都必须经过合规审查
- 分级审核机制：低风险内容由合规专家审核；由合规经理审查的中到高风险内容；总法律顾问审查的主要营销活动
- 审核覆盖全渠道：线上广告、线下素材、社交媒体内容、KOL协作脚本、直播话题
- 出具书面审核意见并留存审核记录供审核

### 第 3 步：发表后监测和预警

- 内容发布后持续监控：广告投诉、平台警示、舆情监控
- 构建关键词监控库：自动检测发布内容中的违规关键词
- 竞争对手合规监控：跟踪竞争对手营销合规活动，以避免行业溢出风险
- 12315热线投诉举报应急预案

### 第四步：违规应急响应

- 发现违规内容：2小时内下架->24小时内出具整改报告->72小时内完成全面审核
- 收到监管通知：立即启动应急预案→法务主导响应→配合调查主动整改
- 媒体曝光/舆情危机：合规+公关+法律三方协同，统一消息传递，快速响应
- 事件后审查：根本原因分析、流程改进、审查清单更新、全公司通知

### 第五步：合规能力建设

- 每季度合规培训：涵盖所有面向客户的部门——营销、销售、电子商务、内容运营
- 年度合规审计：全面审查所有有效营销材料的合规性
- 合规案例库更新：持续收集行业执法案例和内部违规事件
- 合规政策迭代：根据监管变化和运营经验，不断完善内部合规政策
