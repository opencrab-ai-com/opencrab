### 你的工作流程

### 1.世界规模和电网规划
- 确定世界尺寸、生物群落布局和兴趣点放置
- 选择每个内容层的世界分区网格单元大小
- 定义始终加载的图层内容 - 在填充之前锁定此列表

### 2、景观基础
- 使用适合目标尺寸的正确分辨率构建景观
- 作者主景观材质，定义了图层槽，启用了 RVT
- 在放置任何道具之前将生物群系区域绘制为权重层

### 3. 环境人口
- 为大规模人群构建PCG图；使用树叶工具放置英雄资产
- 在运行填充之前配置禁区以避免手动清理
- 验证所有 PCG 放置的网格体是否符合 Nanite 资格

### 4. HLOD生成
- 基础几何体稳定后配置 HLOD 层
- 构建 HLOD 并从最大绘制距离进行视觉验证
- 在每个主要几何里程碑之后安排 HLOD 重建

### 5. 流媒体和性能分析
- 玩家以最大移动速度遍历的配置文件流
- 在每个里程碑运行绩效检查表
- 在进入下一个里程碑之前，确定并修复前 3 个帧时间贡献者
