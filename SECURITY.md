# Security Policy

## Supported Scope

当前仓库主要面向本地运行场景，安全问题优先关注：

- 本地数据泄露
- 渠道 secret 暴露
- 任意文件下载 / 读取
- 渠道鉴权绕过
- 会话或任务的越权访问

## Disclosure

如果你发现安全问题，请不要直接公开提交可利用细节。

当前建议私下联系：

- `sky@opencrab-ai.com`

请尽量附带：

- 影响范围
- 复现步骤
- 预期结果与实际结果
- 是否需要本地特殊配置

## 当前安全边界说明

OpenCrab 当前仍是单机、本地优先形态：

- 没有正式多用户账号体系
- 渠道 secret 当前主要保存在环境变量或本地运行目录
- 本地运行目录默认不进入仓库

更详细的数据边界说明见：

- [隐私与数据边界](./docs/privacy-and-data.md)
